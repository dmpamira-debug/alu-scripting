API Advanced
This project focuses on querying the Reddit API using Python and the Requests module.

Requirements
Python 3.4.3 on Ubuntu 14.04 LTS
PEP 8 style
Requests module for HTTP requests
Tasks
0-subs.py - Returns number of subscribers for a subreddit
1-top_ten.py - Prints titles of first 10 hot posts for a subreddit
2-recurse.py - Recursively returns all hot article titles for a subreddit
3-count.py - Recursively counts and prints sorted keyword occurrences in hot articles
